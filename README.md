## ✨ Features

### 🧠 ML Playground
Run real machine learning predictions directly in the browser — no backend, no API calls, pure client-side inference.
- Upload a trained model exported as a `.json` file
- Currently supports **Decision Tree** and **Logistic Regression** models
- Export your `scikit-learn` model using `sklearn-json` (or a custom JSON exporter) and upload it to get live predictions
- 🔄 **Actively being updated** — Random Forest, XGBoost, SVM, and more models are planned so any sklearn model can be tested live

### 📊 Power BI — Live Reports
Paste any Power BI **"Publish to Web"** link, and the report renders live and interactively inside the page — no static screenshots.

### 📈 Tableau — Live Preview
Paste a Tableau **public share link**, and it loads instantly as a live, interactive view — same idea as Power BI integration.

### ➕ Submit a New Project
A built-in form to add new projects (Power BI, Tableau, or general project type) directly through the site — no code changes required to grow the portfolio.
